# 기술 스택

## 언어 및 프레임워크
- **언어**: C# 10.0
- **프레임워크**: .NET 10.0 for Windows 7.0
- **UI 프레임워크**: Windows Forms
- **인코딩**: UTF-8

## 주요 라이브러리 및 패키지
- **NetOffice.PowerPoint** (1.7.4.11) - PowerPoint Interop
- **NetOffice.Word** (1.7.4.11) - Word Interop  
- **MySql.Data** (9.1.0) - MySQL 데이터베이스 연결
- **MySqlConnector** (2.4.0) - MySQL 커넥터
- **System.Data.SQLite.Core** (1.0.119) - SQLite 데이터베이스

## 데이터베이스
- SQLite (로컬 데이터베이스)
- MySQL/MariaDB (옵션)

## 빌드 플랫폼
- AnyCPU
- x64

## 특수 설정
- AllowUnsafeBlocks: True
