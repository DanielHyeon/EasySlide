8. 멀티 모니터에서 프로그램이 제대로 보이지 않는 현상 수정
  - 모니터 옵션에서 모니터 선택시 다른 모니터에 보이는 현상 수정
  - None 으로 설정시 보여지지 않아야 하는데 보이는 현상 수정

    8.1 요건
	  8.1.1 모니터에 프리젠테이션 화면이 표시가 되어야 한다.
	    8.1.1 싱글모니터인경우 싱글모니터에 프리젠테이션 화면이 나와야 한다.
	    8.1.2 멀티모니터인경우 멀티모니터에 프리젠테이션 화면이 나와야 한다.
		  8.1.2.1 멀티 모니터를 선택하여 선택된 모니터에 프리젠테이션 화면이 나와야 한다.
		  8.1.2.2 멀티 모니터인 경우에도 싱글모드로 설정된 경우에 PRIMARY 화면에 프리젠테이션 화면이 나와야 한다.
	    8.1.3 수동으로 화면 영역을 지정하여 그 영역에 프리젠테이션 화면이 나와야 한다. 
	 8.2.2 프리젠테이션 화면을 제어 해야 한다.
 
	8.2. 조건
	  8.2.1 싱글모니터인 경우
	    8.2.1.1 싱글모니터인지 멀티 모니터인지 확인이 필요
	    8.2.1.1.1 싱글모니터 인지 여부 변수 설정하여 저장
		 
		8.2.1.2 싱글모니터인 경우 싱글모니터에 프리젠테이션 화면이 표시 된다.
	  8.2.2 멀티모니터인경우
		8.2.1.1 멀티 모니터인지 확인이 필요
		  8.2.1.1.1 멀티모니터 인지 여부 변수 설정하여 저장
	    8.2.1.2 멀티 모니터를 사용자가 선택하여 이 값을 저장 한다.  
		  8.2.1.2.1 선택된 모니터에 프리젠테이션이 화면에 표시된다.

	  8.2.3 멀티모니터 인데 싱글모니터 모드로 설정한 경우
	    8.2.3.1 PRIMARY 모니터에 프리젠테이션이 화면에 표시된다.
	   
	  8.2.4 예외사항
		8.2.4.1 세컨더리 화면에 무조건 화면을 표시 하도록 한 경우
	      8.2.4.1.1 세컨더리 화면에 프리젠테이션이 화면에 표시되는 여부  변수에 저장
		  8.2.4.1.2 세컨더리 화면에 프리젠테이션이 화면에 표시 된다.
		 
		8.2.4.2 수동으로 화면 영역을 지정한경우
		  8.2.4.2.1 수동으로 화면 영역을 잡은 경우 변수 저장
		  8.2.4.2.2 수동으로 화면 영역을 잡은 경우 그 영역에 프리젠테이션이 화면에 표시된다.

	수동으로 해상도 표시한 경우에도 제대로 나오는지 확인
